## Controls
- **WASD/Arrows** : Move & Jump(Spacebar works too)
- **F**  			: Freeze/Unfreeze Time

Controls when time is frozen:

- **Left mouse click and drag** : Draw Capture frame
- **Q/E** 					  : Rotate frame
- `**Scroll Wheel**              : Scale Frames
